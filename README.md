## Парсинг блока из блокчейна Bitcoin
<br>Выполнил Гайникамалов Равиль Рашидович

## Текст задания

Есть блокчейн Биткоин (BTC), каждые 15-30мин выходит новый блок с транзакциями, размер блока менее 1 Мб.

Необходимо скачать 1 блок, изучить его структуру (согласно документации в интернете), распарсить его - сохранить данные в виде таблицы в csv файл.
Язык программирования C# .NET.

Информация о блоке Биткоина состоит из основной информации о самом блоке (хэш, высота, индекс и т.п.). Но большую часть занимает список транзакций с этим блоком, их входы и выходы. Как раз это и составляет большое количество вложенных данных внутри блока.


## О проекте

Проект представлен в виде консольного приложения .NET Framework, разработано в IDE Visual Studio 2022.
<br><br>
Приложение скачивает последний блок вышедший блок из блокчейна Биткоина (поэтому нет информации о следующем блоке).После чего получает информацию о нем в json-формате и сохраняет ее в файле latestBlock.json. Далее приложение конвертирует json-файл в csv-файл Outputs.csv, учитывая все вложенные json-объекты, которые представляют транзакции. То есть при парсинге сохранилась структура и порядок вложенности json-объектов.
<br><br>
Файлы latestBlock.json и Output.csv хранятся в папке по адресу: BlockProject\bin\Debug.

## Руководство пользователя

При запуске консольного приложения нужно немного подождать ответа на запрос загрузки информации и блоке. После этого выведется статусе успешной загрузки последнего блока Биткоина и основные данные (хэш, высота, время появления). Также выведется сообщение о сохранении данных в json-файл, а после парсинга из json в csv о сохранении данных в csv-файл. 
<br>Скачанные файлы хранятся в папке проекта по адресу: BlockProject\bin\Debug.
